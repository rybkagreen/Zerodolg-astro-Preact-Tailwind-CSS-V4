# Компонент OptimizedImage

Компонент для отображения оптимизированных изображений с поддержкой современных форматов (AVIF, WebP) и стандартных методов оптимизации производительности.

## Назначение

Компонент реализует лучшие практики оптимизации изображений для веб-сайта юридической компании ZeroDolg, включая:

- Поддержку современных форматов (AVIF, WebP)
- Lazy loading для изображений вне области видимости
- Атрибуты производительности (decoding, sizes)
- Резервные изображения (fallback) для старых браузеров

## Использование

```astro
---
import OptimizedImage from '@/shared/ui/OptimizedImage/OptimizedImage.astro';
---

<OptimizedImage
  src="/images/team/mashulia.webp"
  alt="Масхулиа Леван Зурабович - Арбитражный управляющий"
  width={400}
  height={400}
  loading="lazy"
  decoding="async"
  class="rounded-full object-cover"
/>
```

## Параметры

| Параметр | Тип                         | По умолчанию | Описание                                                   |
| -------- | --------------------------- | ------------ | ---------------------------------------------------------- |
| src      | string                      | -            | Путь к исходному изображению                               |
| alt      | string                      | -            | Описание изображения для SEO и доступности                 |
| width    | number                      | -            | Ширина изображения                                         |
| height   | number                      | -            | Высота изображения                                         |
| loading  | 'lazy' \| 'eager'           | 'lazy'       | Стратегия загрузки изображения                             |
| decoding | 'async' \| 'sync' \| 'auto' | 'async'      | Стратегия декодирования изображения                        |
| class    | string                      | ''           | CSS классы                                                 |
| sizes    | string                      | -            | Атрибут sizes для responsive изображений                   |
| priority | boolean                     | false        | Приоритет загрузки (для изображений выше первой видимости) |

## Особенности реализации

1. Компонент использует тег `<picture>` с поддержкой форматов AVIF и WebP
2. Автоматически создает fallback на оригинальный формат (JPEG/PNG)
3. Реализует lazy loading для отложенной загрузки изображений
4. Включает атрибут decoding для улучшения производительности
5. Учитывает параметр priority для критичных изображений выше первой видимости

## Примеры использования

### Критичное изображение (выше первой видимости)

```astro
<OptimizedImage
  src="/images/hero-background.avif"
  alt="Профессиональная помощь в банкротстве физических лиц"
  width={1200}
  height={800}
  priority
  class="w-full h-full object-cover"
/>
```

### Изображение с lazy loading

```astro
<OptimizedImage
  src="/images/team/briancev.webp"
  alt="Мария Брианцева - юрист по банкротству"
  width={300}
  height={300}
  loading="lazy"
  decoding="async"
  class="rounded-lg"
/>
```

## SEO и доступность

1. Всегда указывайте содержательный атрибут `alt`
2. Используйте релевантные ключевые слова в `alt`-тексте
3. Указывайте правильные размеры для предотвращения сдвига макета (CLS)
