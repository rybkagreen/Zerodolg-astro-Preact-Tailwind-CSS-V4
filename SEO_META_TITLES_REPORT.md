## 💡 ДОПОЛНИТЕЛЬНЫЕ РЕКОМЕНДАЦИИ

### 1. Общие принципы SEO-тайтлов для проекта

#### Структура идеального Title:

```
[Основной запрос] [Город] [Уточнение/УТП] | [Бренд]
```

**Примеры**:

- ✅ `Банкротство физических лиц Москва: списание долгов 2025 | ZeroDolg`
- ✅ `Реструктуризация долгов физлиц Москва от 89000₽ | ZeroDolg`
- ❌ `💰 Лучшее банкротство в Москве - помощь профессионалов!!!` (эмодзи, вода, спам)

#### Длина Title:

- **Яндекс**: 50-65 символов (оптимально)
- **Google**: 50-60 символов (оптимально)
- **Максимум**: 70 символов (может обрезаться)

#### Использование символов:

- ✅ Цифры (2024, 50000₽, 98%)
- ✅ Знаки препинания (: - |)
- ❌ Эмодзи в Title (💰 ⚡) - лучше в Description
- ✅ Символ рубля (₽)

### 2. Description - правила для всех страниц

#### Оптимальная длина:

- **Яндекс**: 140-160 символов
- **Google**: 150-160 символов

#### Структура Description:

```
[Краткое УТП] ⚡ [Преимущество 1] ⚡ [Преимущество 2] ⚡ [Преимущество 3] ⚡ [CTA или дополнительная информация]
```

**Пример для главной**:

```
Списание долгов под ключ от 50000₽ за 4 месяца ⚡ Опыт 15 лет ⚡ 98% успешных дел ⚡ Остановим проценты и приставов ⚡ Консультация бесплатно
```

### 3. Keywords - актуальность в 2024

**Важно**: Яндекс и Google официально не используют meta keywords для ранжирования, НО:

- ✅ Полезны для внутренней аналитики
- ✅ Некоторые региональные поисковики их учитывают
- ✅ Помогают структурировать семантику

**Рекомендация**: Оставить keywords, но не тратить много времени на их оптимизацию.

### 6. A/B тестирование Title

**Инструменты для тестирования**:

1. **Яндекс.Вебмастер** - следить за CTR в поиске
2. **Google Search Console** - анализ кликов
3. **Яндекс.Метрика** - отслеживание источников трафика

**Метрики для оценки**:

- CTR в выдаче (целевой: 5-10% для коммерческих запросов)
- Время на сайте с органического трафика
- Конверсия из органики

**План A/B теста**:

