### 5. ✅ Review Schema компонент

**Файл:** `src/shared/seo/ReviewSchema.astro` (уже существует)

**Функционал:**

- Schema.org Review markup
- Поддержка множественных отзывов
- AggregateRating для организации
- Индивидуальные рейтинги для каждого отзыва

**Применение:**

```astro
<ReviewSchema
  reviews={[
    {
      author: 'Иван Петров',
      rating: 5,
      reviewBody: 'Отличная помощь...',
      datePublished: '2024-09-15',
    },
  ]}
  aggregateRating={{
    ratingValue: '4.9',
    reviewCount: '127',
  }}
/>
```

**SEO эффект:**

- Звездочки рейтинга в сниппетах Google
- Увеличение CTR на 20-30%
- Повышение доверия пользователей
