// При клике по телефону
ym(103604926, 'reachGoal', 'phone_click', {

});

// При расчете калькулятора
ym(103604926, 'reachGoal', 'calculator_result', {

## ✅ Проверка работы целей

### Через DebugView (Real-time):

1. Откройте сайт в новой вкладке
2. В Яндекс.Метрике перейдите: **Отчеты** → **Мониторинг** → **Вебвизор**
3. Выполните действие (отправьте форму)
4. В течение 1-2 секунд должно появиться событие

### Через отчеты (с задержкой до 3 часов):

1. **Отчеты** → **Стандартные отчеты** → **Конверсии** → **Цели**
2. Выберите период
3. Проверьте количество достижений каждой цели
