## 🚀 Подготовка к тестированию

### Запуск автоматизированных тестов

```powershell
# Запустить staging сервер и тесты
.\scripts\test\run-staging-tests.ps1 -StartServer -Verbose

# Или если сервер уже запущен
.\scripts\test\run-staging-tests.ps1 -Verbose
```

### URL для тестирования

- **Staging URL:** http://localhost:3000
- **Lighthouse CI:** http://localhost:9001

### Инструменты для тестирования

- **Chrome DevTools** (F12)
- **Lighthouse** (встроен в Chrome)
- **WAVE** (Web Accessibility Evaluation Tool)

## 🎨 UI/UX тестирование

### Визуальное качество

- [ ] **Шрифты:**
  - [ ] Загружаются корректно
  - [ ] Читаемы на всех размерах экрана
  - [ ] Не слишком мелкие (минимум 16px для основного текста)
- [ ] **Цвета:**
  - [ ] Соответствуют брендингу
  - [ ] Достаточный контраст (минимум 4.5:1)
  - [ ] Ошибки отображаются красным
  - [ ] Успех - зеленым
- [ ] **Изображения:**
  - [ ] Все изображения загружаются
  - [ ] Alt-тексты присутствуют
  - [ ] Качество достаточно высокое
  - [ ] Размеры оптимизированы для web

### Интерактивность

- [ ] **Ховер эффекты:**
  - [ ] Кнопки реагируют на наведение
  - [ ] Ссылки изменяются при ховере
  - [ ] Анимации плавные
- [ ] **Фокус:**
  - [ ] Видимые индикаторы фокуса
  - [ ] Порядок табуляции логичен
- [ ] **Загрузка:**
  - [ ] Индикаторы загрузки (спиннеры) работают
  - [ ] Нет блокировок интерфейса

### Адаптивность

- [ ] **Desktop (>1200px):**
  - [ ] Контент занимает экран оптимально
  - [ ] Все элементы видны
- [ ] **Laptop (992-1199px):**
  - [ ] Адаптация корректная
- [ ] **Tablet (768-991px):**
  - [ ] Меню адаптируется
  - [ ] Контент перестраивается
- [ ] **Mobile (≤767px):**
  - [ ] Мобильное меню работает
  - [ ] Формы удобны для заполнения
  - [ ] Кнопки достаточно большие (минимум 44px)
