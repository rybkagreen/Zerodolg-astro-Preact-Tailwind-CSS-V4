# Финальное исправление контрастности паттернов

_Дата: 01.10.2025_

## 🎯 Проблема

Фоновые акварельные паттерны были слишком контрастными и мешали чтению контента.
Требовалось сделать их едва заметными.

## ✅ Решение

Установлена максимально низкая прозрачность `opacity: 4%` для всех фоновых
паттернов.

## 🎨 Техническое решение

### CSS изменения

```css
/* До */
opacity-12 /* или opacity-10, opacity-25, opacity-30 */

/* После */
opacity-[0.04] /* 4% прозрачность */
```

### Особенности применения

- Использована нотация Tailwind `opacity-[0.04]` для точного контроля
- Сохранен `mix-blend-multiply` для естественного смешивания
- Все остальные свойства (`background-size: cover`,
  `background-position: center`) не изменились

- 🎨 **Сохранена эстетика** - акварельные паттерны добавляют изысканности
- ⚡ **Единообразие** - все секции имеют одинаковую контрастность (4%)

### Пользовательский опыт
