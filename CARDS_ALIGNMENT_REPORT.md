# Отчет о выравнивании карточек сравнения

_Дата: 01.10.2025_

## 🎯 Цель

Выровнять все элементы в карточках секции сравнения банкротства и
реструктуризации для одинаковой высоты и правильного расположения элементов.

## 🔧 Проблемы до исправления

- Карточки имели разную высоту
- Кнопки располагались на разных уровнях
- У карточки банкротства есть badge "Популярный выбор", который увеличивал
  высоту
- У карточки реструктуризации две кнопки, у банкротства - одна

## ✅ Применённые решения

### 1. Flexbox Layout

Добавлена flexbox структура для обеих карточек:

```css
flex flex-col h-full
```

### 2. Структурирование контента

Разделен контент на области:

- **Фиксированная область**: Header, icon, pricing, stats, advantages,
  considerations
- **Гибкая область**: "Подходит" блок с `flex-grow`
- **Фиксированная нижняя область**: Кнопки с `mt-auto`

### 3. Выравнивание кнопочной области

**Банкротство**:

```astro
<div class='mt-auto space-y-3'>
  <button>Консультация по банкротству</button>
  <div class='h-8'></div>
  <!-- Placeholder -->
</div>
```

**Реструктуризация**:

```astro
<div class='mt-auto space-y-3'>
  <button>Консультация по реструктуризации</button>
  <a href='/restrukturizaciya-dolgov'>Подробнее →</a>
</div>
```

### 4. Badge выравнивание

Добавлен невидимый placeholder badge для карточки реструктуризации:

```astro
<div class='invisible absolute -top-3 ...'>Placeholder</div>
```

## 🎨 Техническая реализация

### Flexbox структура карточек:

```css
/* Основная карточка */
.group.flex.flex-col.h-full

/* Контентная область */
.relative.z-10.pt-4.flex.flex-col.flex-grow

/* Гибкая область контента */
.flex-grow

/* Фиксированные кнопки внизу */
.mt-auto.space-y-3
```

### Выравнивание высоты:

- `h-full` - карточки занимают полную доступную высоту
- `flex-grow` - средняя область растягивается
- `mt-auto` - кнопки прижимаются к низу
