## Как внести вклад

Мы приветствуем вклад в развитие проекта! Чтобы внести свой вклад:

1. Форкните репозиторий
2. Создайте ветку для вашей функции
3. Внесите изменения
4. Напишите тесты
5. Обновите документацию
6. Создайте Pull Request

### Рекомендации по внесению вклада

1. Следуйте руководству по стилю кода
2. Пишите понятные комментарии и документацию
3. Тестируйте свои изменения
4. Соблюдайте архитектурные принципы проекта
5. Уважайте других участников проекта
