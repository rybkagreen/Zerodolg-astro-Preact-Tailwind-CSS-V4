# Улучшения виджетов - Сводка изменений

## Дата: 2025-10-03

### 1. Виджет Bitrix Callback - Современный дизайн

#### Изменения в кнопке виджета:

- ✅ **Увеличен размер**: с 14×14 до 16×16 (с `w-14 h-14` до `w-16 h-16`)
- ✅ **Градиентный фон**: добавлен красивый градиент
  `from-blue-500 via-blue-600 to-purple-600`
- ✅ **Улучшенные hover-эффекты**:
  - Градиент становится темнее при наведении
  - Масштабирование `hover:scale-110`
  - Активное нажатие `active:scale-95`
- ✅ **Улучшенная тень**:
  - Увеличена тень `shadow-2xl`
  - Добавлена анимированная тень с пульсацией через `@keyframes pulse-ring`
  - Custom box-shadow с цветовым свечением
- ✅ **Анимация пульсации**: улучшенная анимация с градиентом на animate-ping
  элементе
- ✅ **Увеличенная иконка**: размер SVG с 6×6 до 7×7 с drop-shadow

#### Изменения в popup форме:

- ✅ **Увеличенная ширина**: с `w-80` (320px) до `w-96` (384px)
- ✅ **Больший padding**: с `p-6` до `p-8`
- ✅ **Улучшенные скругления**: с `rounded-lg` до `rounded-2xl`
- ✅ **Усиленная тень**: custom box-shadow с большей глубиной
- ✅ **Увеличенный backdrop blur**: с `backdrop-blur-sm` до `backdrop-blur-md`
- ✅ **Темнее фон overlay**: с `bg-black/50` до `bg-black/60`

#### Изменения в элементах формы:

- ✅ **Заголовок**:
  - Увеличен размер с `text-lg` до `text-xl`
  - Добавлен градиентный текст (blue-purple)
  - Добавлен font-bold
- ✅ **Кнопка закрытия**:
  - Градиентный hover-эффект (red-pink)
  - Масштабирование при наведении
  - Цветная иконка (красная)
- ✅ **Поля ввода**:
  - Увеличен padding с `px-3 py-2` до `px-4 py-3`
  - Скругления с `rounded-md` до `rounded-xl`
  - Увеличена граница с `border` до `border-2`
  - Более яркий focus ring (blue-400)
  - Добавлен hover-эффект на границе
- ✅ **Кнопка отправки**:
  - Градиентный фон (blue-purple)
  - Увеличен padding
  - Улучшенная тень с градиентом
  - Эффекты масштабирования
  - Добавлена emoji иконка 📞

### 2. Стикер онлайн посетителей - Стеклянный эффект

#### Основные изменения:

- ✅ **Полупрозрачный фон**:
  - Изменен с непрозрачного градиента на `rgba(16, 185, 129, 0.85)`
  - Добавлен glass effect с `backdrop-filter: blur(12px) saturate(180%)`
  - Поддержка webkit для Safari
- ✅ **Тонкая граница**: добавлена белая полупрозрачная граница
  `rgba(255, 255, 255, 0.2)`
- ✅ **Улучшенная тень**:
  - Уменьшена интенсивность до 0.25
  - Добавлена вторая тень для границы
- ✅ **Оптимизация размеров**:
  - Padding уменьшен с `12px 16px` до `10px 14px`
  - Размер шрифта с 14px до 13px
  - Font-weight увеличен до 600

#### Изменения для мобильных устройств:

- ✅ **Позиционирование**:
  - Размещение в **левом нижнем углу** (`left: 12px`)
  - Удалено растягивание на всю ширину (`width: auto`)
  - Убрано `right: auto`
- ✅ **Уменьшенный размер**:
  - `max-width: 180px` (было 280px)
  - `font-size: 12px` (было 13px)
  - `padding: 8px 12px` (было 10px 14px)
- ✅ **Отступ снизу**: изменен с 100px на 90px (над кнопкой callback)
- ✅ **Иконки и элементы**:
  - Пульсирующая точка: 6×6px на мобильных (было 8×8)
  - Иконка: 14×14px на мобильных (было 16×16)
  - Добавлено `flex-shrink: 0` для предотвращения сжатия

#### Дополнительные улучшения:

- ✅ **Text shadow**: добавлена легкая тень на счетчик для лучшей читаемости
- ✅ **Box shadow на пульсации**: добавлено свечение белой точки
- ✅ **Font-weight счетчика**: увеличен с 600 до 700

### 3. CSS Анимации

#### Новая анимация для кнопки Bitrix Callback:

```css
@keyframes pulse-ring {
  0% {
    box-shadow:
      0 10px 40px rgba(59, 130, 246, 0.4),
      0 0 0 0 rgba(59, 130, 246, 0.5);
  }
  50% {
    box-shadow:
      0 12px 45px rgba(59, 130, 246, 0.5),
      0 0 0 8px rgba(59, 130, 246, 0);
  }
  100% {
    box-shadow:
      0 10px 40px rgba(59, 130, 246, 0.4),
      0 0 0 0 rgba(59, 130, 246, 0);
  }
}
```
