# Краткая справка по командам расширений Qwen Code

## astro-dev

| Команда | Описание |
|---------|----------|
| `/dev` | Запустить сервер разработки Astro |
| `/build` | Собрать проект Astro для продакшена |
| `/preview` | Предварительный просмотр сборки |
| `/test` | Запустить тесты проекта |
| `/typecheck` | Проверить типы TypeScript |
| `/lint` | Запустить линтер |

## git-helper

| Команда | Описание |
|---------|----------|
| `/status` | Показать статус репозитория Git |
| `/add` | Добавить все изменения в индекс Git |
| `/commit` | Создать коммит с сообщением |

## deps-manager

| Команда | Описание |
|---------|----------|
| `/install` | Установить все зависимости проекта |
| `/update` | Обновить зависимости проекта |

## perf-check

| Команда | Описание |
|---------|----------|
| `/audit` | Запустить аудит производительности с помощью Lighthouse |

## Особенности использования

1. Все команды доступны сразу после перезапуска Qwen Code
2. При конфликтах имен используйте префикс расширения: `/astro-dev.dev`
3. Некоторые команды могут запрашивать дополнительные параметры (например, `/commit` запрашивает сообщение коммита)
