# SpecialOffers Component

## Описание
Компонент SpecialOffers реализует элементы срочности и социальное доказательство для повышения конверсии на сайте юридических услуг по банкротству. Компонент включает в себя несколько подкомпонентов:

- Таймер обратного отсчета
- Уведомления о действиях пользователей (социальное доказательство)
- Счетчик активных посетителей
- Калькулятор экономии при банкротстве
- Баннер специальных предложений
- Бейджи доверия
- Счетчик доступных мест

## Структура компонента
```
SpecialOffers/
├── SpecialOfferBanner
├── SocialProofNotifications
├── LiveVisitorsCounter
├── SavingsCalculator
├── TrustBadges
└── SpotsCounter
```

## Использование
Компонент автоматически инициализируется при рендеринге страницы. Для использования компонента достаточно импортировать его в нужный файл:

```astro
---
import { SpecialOffers } from '../components/islands/SpecialOffers.astro';
---

<SpecialOffers />
```

## Подкомпоненты

### SpecialOfferBanner
Баннер специальных предложений с таймером обратного отсчета. Отображается в правом нижнем углу экрана. Может быть закрыт пользователем, после чего не отображается в течение сессии.

### SocialProofNotifications
Уведомления о действиях других пользователей. Отображаются в правом верхнем углу экрана с интервалом в 20 секунд.

### LiveVisitorsCounter
Счетчик активных посетителей страницы. Обновляется каждые 30 секунд.

### SavingsCalculator
Калькулятор экономии при банкротстве. Позволяет пользователю указать сумму долга и показывает потенциальную экономию.

### TrustBadges
Бейджи доверия, отображающие сертификаты и гарантии безопасности.

### SpotsCounter
Счетчик доступных мест для участия в специальном предложении.

## Стилизация
Все стили компонента находятся в файле `src/styles/03-components/_special-offers.css` и следуют методологии BEM в рамках архитектуры ITCSS.

## Зависимости
- Preact hooks (useState, useEffect)

## Совместимость
Компонент совместим с Astro и использует islands architecture для оптимизации производительности.
