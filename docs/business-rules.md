# Бизнес-правила юридического сайта ZeroDolg

## Общие принципы

1. **Точность информации**: Вся юридическая информация должна быть точной и актуальной
2. **Доступность**: Сложные юридические понятия должны объясняться простым языком
3. **Конфиденциальность**: Все данные клиентов должны обрабатываться с соблюдением требований по защите персональных данных
4. **Профессионализм**: Все материалы должны соответствовать профессиональным стандартам юридического сообщества

## Требования к контенту

### Обязательные уведомления
- Политика конфиденциальности
- Условия использования
- Отказ от ответственности
- Информация о лицензиях и сертификатах

### Структура юридических услуг
- Четкое описание услуги
- Этапы работы
- Примерная стоимость
- Сроки выполнения
- Результат

## SEO-стратегия для юридических услуг

### Ключевые слова
- "банкротство физических лиц"
- "банкротство юридических лиц"
- "освобождение от долгов"
- "реструктуризация долгов"

### Геотаргирование
- Региональные запросы ("банкротство в Москве", "банкротство в СПб")

### Контент-стратегия
- Часто задаваемые вопросы
- Блог с правовыми комментариями
- Интерактивный калькулятор для оценки ситуации

## Правила взаимодействия с клиентами

### Формы заявок
- Формы обратной связи (звонок, консультация, банкротство)
- Поддержка различных типов запросов: callback, consultation, bankruptcy, calculator
- Валидация данных формы до отправки в Bitrix24
- Поддержка JSON и FormData запросов

### Ответы на запросы
- Автоматическая отправка данных в Bitrix24 CRM
- Обработка заявок через webhook с таймаутом
- Отслеживание конверсий и аналитика событий
- Поддержка тестового режима для разработки

## Комплаенс требования

### Российское законодательство
- Федеральный закон "О персональных данных"
- Требования к юридическим компаниям
- Реквизиты и лицензии
- Защита персональных данных клиентов

## Особенности реализации сайта

### Технологии
- Astro v5.13.7 с серверным рендерингом (SSR)
- Preact для интерактивных элементов (островная архитектура)
- TypeScript v5.9.2 для типобезопасности
- Tailwind CSS v3.4+ с пользовательскими дизайн-токенами

### Основные функции
- Интерактивный калькулятор для оценки ситуации с долгами
- Интеграция с Bitrix24 CRM через API
- Обработка форм с валидацией и отправкой в CRM
- Многоступенчатый процесс с динамическими рекомендациями
- Прогрессивное улучшение (работает без JavaScript)

### Безопасность
- Защита CSRF
- Правильные заголовки безопасности
- Пользовательская политика безопасности контента (CSP)
- Валидация данных на клиенте и сервере

### Производительность
- Рендеринг на стороне сервера для SEO
- Предзагрузка при скролле для повышения производительности
- Пользовательские заголовки кэширования
- Оптимизация изображений с поддержкой WebP/AVIF
- Дерево-шейкинг CSS

### Интеграции
- Google Analytics для отслеживания пользовательского поведения
- Yandex Metrics для дополнительной аналитики
- Bitrix24 CRM для управления заявками
- Пользовательский виджет обратного звонка с маскированием телефонов
