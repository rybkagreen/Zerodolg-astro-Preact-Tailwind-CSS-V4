# Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Что такое ZeroDolg Astro?
ZeroDolg Astro - это корпоративный сайт юридической компании, специализирующейся на банкротстве физических лиц. Проект построен с использованием современного стека технологий на базе Astro.

### Какие технологии используются в проекте?
Проект использует следующие технологии:
- **Astro** - статический генератор сайтов
- **Preact** - легковесная альтернатива React для интерактивных компонентов
- **TypeScript** - строгая типизация JavaScript
- **CSS** - современный CSS с пользовательскими свойствами
- **Alibaba CMS** - облачная система управления контентом

## Разработка

### Как установить проект?
```bash
npm install
```

### Как запустить локальный сервер разработки?
```bash
npm run dev
```

### Как собрать проект для продакшена?
```bash
npm run build
```

### Какие есть скрипты для разработки?
| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск локального сервера разработки |
| `npm run build` | Сборка проекта для продакшена |
| `npm run preview` | Предварительный просмотр собранного сайта |
| `npm run astro ...` | Запуск команд Astro CLI |

```
zerodolg-astro/

├── public/              # Статические файлы
├── src/
│   ├── components/      # Компоненты

```

## Компоненты

### Как добавить новый компонент?
1. Создайте файл компонента в соответствующей директории:
   - `/src/components/ui/` для UI компонентов
   - `/src/components/sections/` для секций
   - `/src/components/islands/` для интерактивных компонентов
2. Следуйте шаблону компонента из документации
3. Добавьте документацию в начале файла

### Как работает интеграция с Alibaba CMS?
Интеграция осуществляется через сервисы в директории `/src/lib/`:
- `alibaba-cms.js` - низкоуровневое API взаимодействие
- `content-service.js` - сервис для получения контента
- `cms-content-models.js` - модели данных

### Как добавить новый тип контента в CMS?
1. Добавьте модель в `cms-content-models.js`
2. Добавьте функции получения данных в `alibaba-cms.js`
3. Добавьте сервисные функции в `content-service.js`
4. Обновите компоненты для использования новых данных

## Стили

### Какая архитектура CSS используется?
Проект использует архитектуру ITCSS (Inverted Triangle CSS) с методологией BEM для именования классов.

### Как добавить новые стили?
1. Определите, в какой слой ITCSS относятся стили:
   - `00-settings` - переменные и конфигурации
   - `01-generic` - сбросы и нормализация
   - `02-elements` - стили HTML элементов
   - `03-components` - стили UI компонентов
   - `04-sections` - стили секций страниц
   - `05-utilities` - вспомогательные классы
2. Создайте файл стилей в соответствующей директории
3. Подключите файл в `main.css`

- Собственный сервер

## Проблемы и отладка

### Что делать, если компонент не отображается?
1. Проверьте правильность импорта компонента
2. Убедитесь, что компонент находится в нужной директории
3. Проверьте консоль браузера на наличие ошибок
4. Проверьте логи сервера разработки

### Как отладить проблемы с CMS?
1. Проверьте подключение к Alibaba CMS
2. Убедитесь, что API ключ действителен
3. Проверьте структуру данных в CMS
4. Посмотрите логи консоли с префиксом `[Alibaba CMS]`

### Как проверить производительность сайта?
1. Используйте Lighthouse в инструментах разработчика Chrome
2. Проверьте оценку Core Web Vitals
3. Используйте инструменты анализа сети
