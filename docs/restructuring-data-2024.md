# Актуальные данные по реструктуризации долгов - Октябрь 2024

## Ключевые изменения в законодательстве

### Федеральный закон №127-ФЗ "О несостоятельности (банкротстве)" - последние поправки 2024

**Изменения в процедуре реструктуризации:**

- Увеличен максимальный срок реструктуризации с 3 до 4 лет для социально
  значимых категорий
- Снижен минимальный размер дохода для реструктуризации до 1,5 прожиточного
  минимума (был 2 ПМ)
- Введены льготы для пенсионеров и инвалидов при утверждении планов
  реструктуризации

**Новые требования к должникам:**

- Обязательное прохождение финансовой грамотности (8 часов)
- Ежемесячная отчетность перед финансовым управляющим в электронном виде
- Запрет на получение новых кредитов без согласования с управляющим
