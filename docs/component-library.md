## Общие принципы

## Базовые UI компоненты

### Button
- Используется для всех интерактивных элементов
- Варианты: primary, secondary, outline, link
- Всегда имеет текстовую метку или ARIA-описание

### FormField
- Компонент для отдельного поля формы
- Включает label, input, validation message
- Поддерживает все типы input'ов

## Компоненты форм

### CallbackWidget
- Компонент `BitrixCallback.astro` для обратного звонка
- Использует библиотеку `bitrix-callback-widget.ts`
- Включает маскирование телефонов и валидацию
- Интеграция с Bitrix24

### CalculatorForm
- Интерактивный калькулятор в `src/features/calculator/`
- Многоступенчатая форма для оценки долга
- Динамические рекомендации на основе ввода
- Отправка результатов в Bitrix24

## Секционные компоненты

### HeroSection
- Главная секция на главной странице
- Заголовок, подзаголовок, призыв к действию
- Адаптивный фон с градиентами

### ServiceSection
- Секции услуг (банкротство, реструктуризация)
- Иконка, заголовок, описание
- Связь с юридическими аспектами

## Интерактивные компоненты (Islands)

### CalculatorIsland
- Интерактивный калькулятор на основе Preact
- Состояние и логика на клиенте
- Динамические результаты и рекомендации

### TeamInteractive
- Компоненты команды с интерактивными элементами
- Показывает детали при клике
- Использует островную архитектуру Astro

## Макеты (Layouts)

### RootLayout
- Корневой макет для всего сайта
- Включает SEO метатеги
- Обеспечивает структуру страницы

### PageLayout
- Макет для внутренних страниц
- Включает шапку, подвал
- Обеспечивает навигацию

## Виджеты

### HeaderWidget
- Навигационная шапка сайта
- Адаптивное меню
- Кнопки для форм

### FooterWidget
- Подвал сайта
- Контактная информация
- Ссылки на политику и т.д.

### FAQWidget
- Вопросы-ответы
- Интерактивный аккордеон
- Поддержка Rich Content

### ReviewsWidget
- Отзывы клиентов
- Слайдер с отзывами
- Рейтинги и изображения

## Типографика

### Заголовки
- h1: 2.5rem, bold - для основного заголовка страницы
- h2: 2rem, semibold - для секций
- h3: 1.5rem, semibold - для подсекций
- h4: 1.25rem, medium - для элементов контента

### Основной текст
- Основной: 1rem, 400 - для основного контента
- Подзаголовок: 1.125rem, 500 - для дополнительной информации
- Мелкий: 0.875rem, 400 - для вспомогательного текста

## Цветовая палитра

### Основные цвета
- primary: #1e40af (индиго для акцентов)
- secondary: #64748b (нейтральный синий)
- success: #10b981 (зеленый для успеха)
- warning: #f59e0b (оранжевый для предупреждений)
- error: #ef4444 (красный для ошибок)

### Фоновые цвета
- background: #f8fafc (светло-серый)
- surface: #ffffff (белый)
- muted: #f1f5f9 (приглушенный серый)

## Доступность (a11y)

### ARIA-атрибуты
- Все интерактивные элементы имеют role и aria-* атрибуты
- Формы содержат корректные label и aria-describedby
- Кнопки и ссылки имеют aria-label или aria-labelledby

### Контрастность
- Текст и фоны соответствуют WCAG 2.1 AA (коэффициент 4.5:1)

- Повторное использование компонентов
