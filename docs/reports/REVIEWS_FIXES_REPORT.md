## 3. ✅ Исправлена проблема с высотой карусели

### Изменения:
- Изменен `overflow` с `visible` на `hidden` для предотвращения выхода контента
- Увеличена минимальная высота wrapper до 550px (580px на десктопе)
- Добавлен паддинг для слайдов (0.5rem) для компенсации эффектов hover
- Уменьшен масштаб при hover с 1.02 до 1.01
- Добавлено ограничение максимальной высоты карточек (550px)

### Технические детали:
```css

  overflow: hidden; /* Скрываем выходящий контент */

  max-height: 550px; /* Ограничение высоты */

  transform: translateY(-4px) scale(1.01); /* Уменьшен масштаб */
  z-index: 10; /* Поднимаем выше других карточек */
}
```
