2. **НИКОГДА не использовать !important** в CSS
3. **Проверять существующие CSS файлы** перед добавлением стилей
4. **Следовать BEM методологии** для именования классов
5. **Использовать data-атрибуты** для JavaScript вместо стилей
6. **Добавлять JSDoc комментарии** к компонентам для автоматической генерации документации
7. **Следовать структуре документации** при создании новых файлов
