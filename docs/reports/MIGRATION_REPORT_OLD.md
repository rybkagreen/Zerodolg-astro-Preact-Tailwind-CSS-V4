## 🚀 Ключевые преимущества Astro

1. **Partial Hydration** - JavaScript загружается только для интерактивных компонентов
2. **Islands Architecture** - Изоляция интерактивности
3. **Zero JS by default** - Статические компоненты не требуют JS
4. **Built-in оптимизации** - Автоматическая оптимизация изображений, CSS, JS
5. **Component Agnostic** - Можно использовать React, Vue, Svelte, Preact
